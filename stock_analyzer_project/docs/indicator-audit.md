# 技术指标口径核查记录

核查日期：2026-06-09

## 结论

基础指标现统一由 `indicator_engine.py` 计算。主分析、历史信号回测、统一组合回测和高级信号模块不再各自维护一套基础公式。

## 差异与处理

| 指标 | 原有差异 | 统一结果 |
|---|---|---|
| MA/VOL_MA | 主分析仅部分周期，回测另算 5/10/20/50/200 | 统一生成 MA 5/10/20/50/200 与 VOL_MA 5/10/20 |
| AO | 使用典型价格和 5/13 周期 | 改为中间价 `(H+L)/2` 的 5/34 SMA |
| BBD | 被描述为主力资金 | 保留原项目公式，但明确为 AO 动量派生项，不代表真实资金流 |
| MACD | 主分析柱体乘 2，组合回测使用另一组列名 | 统一 DIF/DEA/MACD，并将 `MACD_SIGNAL` 设为 DEA 兼容别名 |
| RSI | 主分析和回测均使用简单移动平均 | 改为 Wilder 14 日递归平滑 |
| EMA | 预热期处理不一致 | 统一 `adjust=False`，达到对应周期后输出 |
| KDJ | 主分析的 EWM 参数实际约为 alpha=0.6，回测约为 1/3 | 统一 RSV(9)，K/D alpha=1/3 |
| CCI | 平均绝对偏差计算窗口不一致 | 统一为同一 14 日窗口内典型价格对窗口均值的平均绝对偏差 |
| ATR | 使用 14 日简单平均 | 改为 Wilder 14 日递归平滑 |
| VWAP | 主分析用收盘价，回测用典型价格 | 统一为典型价格累计成交量加权 |
| FIB | 口径基本一致 | 统一为 21 日滚动高低点 |

## 特殊限制

1. 日线 VWAP 是整段日线序列的累计代理，不等同于交易所分钟级或逐笔当日 VWAP。
2. BBD 是项目自定义研究因子，不应命名或解释为真实主力资金净流入。
3. 当前回测属于固定持有期事件研究，不包含手续费、滑点、涨跌停、停牌、资金占用和重复持仓处理。
4. 因此报告只展示真实可得的样本数、胜率和持有期收益，不展示未经策略级资金模拟得到的净值、年化收益或最大回撤。
