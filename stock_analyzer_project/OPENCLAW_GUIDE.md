# LumenAlpha Daily Research Guide

本文件同时定义默认关注股票池，并说明如何生成每日行业研究报告。

## 运行方式

```powershell
python -m stock_analyzer_project.guide_report `
  --guide "stock_analyzer_project\OPENCLAW_GUIDE.md"
```

报告输出至：

```text
stock_analyzer_project/reports/daily/
```

生成两类文件：

- `daily_*.html`：量化指标与信号摘要。
- `research_*.html`：按行业分组的逐股研究报告。

## 研究原则

1. 技术信号只作为证据，不直接等同于投资结论。
2. 综合判断同时考虑行业属性、公司定位、趋势、动量、量能、波动和关键价位。
3. 每只股票必须提供支持因素、反证风险、观察条件和判断失效条件。
4. BBD 是项目自定义 AO 动量派生项，不代表真实主力资金流。
5. 日线 VWAP 是累计典型价格成交量加权代理，不等同于分钟级当日 VWAP。

```powershell
python -m stock_analyzer_project.stock_analyzer_main --indices 000001,399006
python -m stock_analyzer_project.stock_analyzer_main --codes 300750,600519
python -m stock_analyzer_project.stock_analyzer_main --codes 300750 --backtest
```

## 每日邮件

参考 `.env.example` 配置股票、收件地址和 SMTP 授权信息，然后运行：

```powershell
python -m stock_analyzer_project.daily_job
```

SMTP 未配置完整时，任务仍会生成报告，但会跳过邮件发送。
